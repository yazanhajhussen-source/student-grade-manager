# 📊 Student Grade Manager

A Java program to manage and calculate student grades.

## ✨ Features

- Add students with their grades
- View all students with letter grades (A/B/C/D/F)
- Calculate class average
- Find top student

## 🚀 How to Run

```bash
javac StudentGradeManager.java
java StudentGradeManager
```
